Portfolio Details

Project information

  • Project Title: JobBusiness (Android Mobile Application for Local Private Sector Job Listings and Small Business Advertising)
  • Developer: Hifdzullah Matali
  • Year developed: 2019
  • Developed using: Java, XML, Firebase
  • Project Github:Click Here

This project is developed by the author to provides the public information and updates regarding to job listings from private sectors and small business product advertisment. The purpose of this mobile application is to give more opportunities for jobseekers to find job listings within private sector and opporutnities for small businesses to advertise their products.

    Functional Requirements

  1. User can view and read post from job listing page and product advertisment page in real-time data
  2. User can use the search bar and search filter to search for specific search query
  3. Admin can insert new post and delete existing post
  4. Admin can view and read exsiting uploaded post and currently uploaded post in real-time data
  5. User Authentication
